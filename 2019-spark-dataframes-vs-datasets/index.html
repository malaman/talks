Apache Spark Dataframes vs Datasets

Apache Spark Provides

  • Resilient Distributed Dataset (since Spark 0.x)
  • DataFrames = Dataset[Row] (since Spark 1.3)
  • Datasets (since Spark 1.6)

RDD vs Datasets

  • Both RDD and Dataset use case classes
  • Why should i use Dataset?
  • https://jaceklaskowski.gitbooks.io/mastering-spark-sql/spark-sql-dataset-rdd.html

Dataset API vs SQL

  • https://jaceklaskowski.gitbooks.io/mastering-spark-sql/spark-sql-dataset-vs-sql.html

Dataset pros

  • Compile time type checks
  • Easier to understand the code for scala developers
  • Easier to test (because of case classes)

Dataset cons

  • Limited operations in Dataset API
  • More code

Dataframe pros

  • Availability of Dataset API and SQL
  • Code is very concise

Dataset cons

  • No checks on compile time
  • More difficult to maintain

Dataset benefits

  • Compile time type checks
  • Easier to understand the code for scala developers
  • Easier to test (because of case classes)

Dataframe benefits

  • Easier to understand the code for scala developers
  • Easier to test (because of case classes)
